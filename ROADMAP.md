This is the high level roadmap for the Character Builder

# Character Creation
- Supporting all Character Creation Options from the JAZZ Age and Modern Age SRDs

# Additional Eras
- Add All ERAs from the SRDs, starting with Cold War and Victorian

# Extend Play Mode
- Editing for all fields that can be modified during play or downtime activities
- Dice Roller
- Rollable Elements

# Improved Export and Character Sheet
- Being able to Export to a Character Sheet format that looks more like the originals
- Export to text

# Additional Tools
- Adding custom Archetypes

# What Else?
Anything else you'd like to see in the tool? Raise an issue or send a message.
